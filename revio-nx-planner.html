Each cell's uses carry unique barcodes — no carryover clash

Internal planning aid. Styled after the PacBio Revio v13.5 / SPRQ-Nx technical overview; not an official PacBio tool. Packing enforces one rule — no barcode repeats across the uses of a single SMRT Cell — reflecting SPRQ-Nx use-to-use carryover (typically <0.1%). Constants used: 4 cells per tray, 4 wells per plate, up to 3 uses per cell, ~45 min added prep for uses 2–3, and the 108 h (4.5-day) window from first breakout to the start of use 3. Multi-use Nx is validated for library inserts ≥5 kb. Cost figures ($1480 / $888 / $690 per acquisition at 1 / 2 / 3 uses; $995 single-use) are the technical-overview list estimates and exclude instrument amortisation; confirm local pricing. Research use only.
